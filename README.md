# KoinX Tax Harvesting Dashboard

A modern React-based Tax Loss Harvesting dashboard inspired by the KoinX frontend internship assignment.

## 🚀 Live Demo

https://koinx-tax-harvesting-gamma.vercel.app/

---

## ✨ Features

- Modern dark fintech UI
- Responsive dashboard design
- Tax loss harvesting simulation
- Dynamic capital gains calculations
- Selectable crypto holdings
- Sorting functionality
- View All / Show Less holdings
- Real-time harvesting updates
- Reusable React components
- Clean folder architecture

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Icons
- JavaScript

---

## 📊 Functionality

Users can:

- View capital gains before harvesting
- Select crypto assets to harvest
- Dynamically calculate effective capital gains
- View estimated tax savings
- Sort holdings based on short-term and long-term gains/losses
